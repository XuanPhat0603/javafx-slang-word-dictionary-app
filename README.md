## Function

- Danh sách từ lòng
- Tìm kiếm theo SlangWord
- Thêm từ lóng
- Sửa từ lóng
- Xoá từ lóng
- Reset từ điển
- Lịch sử tìm kiếm
- Từ lóng mỗi ngày (5 phút/từ mới)
- Nhìn từ đoán nghĩa
- Nhìn nghĩa đoán từ
